## Prerequisites

### Required Software
- Node.js (v18.x or higher)
- npm (v9.x or higher)
- Docker (v24.x or higher)
- Docker Compose (v2.x)

### Access Requirements
- AWS Account with appropriate IAM permissions
- GitHub repository access
- CI/CD platform credentials (e.g., Jenkins, GitHub Actions)
